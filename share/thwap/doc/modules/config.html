Module config

Dynamic configuration module.

There are no methods, no functions. The config object that is returned is a table with all of the configuration directives inside it. It reads in configration files in the following order:

  1. ~/.thwap/init.d/thwap.config
  2. ~/.thwaprc
  3. /etc/thwap/thwap.config

Stopping at the first result found. It then goes on to do system dependent configuration. In this case we standardize on GNU utilities so we specify the g* variants on the *BSD systems.
